# Slice Completion Checklist

- [ ] Exactly one active brief controlled the work.
- [ ] Expected paths and explicit non-goals were inspected before editing.
- [ ] The narrow new-slice proof was run.
- [ ] Relevant regression checks were run separately.
- [ ] Deterministic evidence was regenerated.
- [ ] Snapshot or generated-file drift check passed.
- [ ] Baseline, scenario, authorization, and applied state remain distinct.
- [ ] No real workforce data or production secrets were introduced.
- [ ] Generated caches, builds, and local environments were removed.
- [ ] Implementation findings state what the evidence does not prove.
- [ ] Cross-repository reconciliation was evaluated without inventing authority.
- [ ] Deferred work and the next operator gate are explicit.
